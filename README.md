# To-Do App

Eine moderne To-Do-Liste, erstellt mit HTML, CSS und JavaScript.

## 🔧 Funktionen

- Aufgaben hinzufügen
- Aufgaben löschen
- Übersichtliches und sauberes Design
- Einfache Bedienung

## 🚀 Live Demo

https://mgeex61.github.io/todo-app/

## 🛠️ Technologien

- HTML
- CSS
- JavaScript

## 📌 Nutzung

1. Aufgabe eingeben
2. Auf "Hinzufügen" klicken
3. Aufgabe wird in der Liste angezeigt
4. Mit ❌ wieder löschen
